# ToDo App

This is a simple console based ToDo app built using C#.

## Getting Started
- Install and Set up Visual Studio 2022 Communoty version
- Clone the repo
- Open the solution file in Visual Studio
- Run the project

## Usage
- Add a new task
- View all tasks
- View completed tasks
- View pending tasks
- Mark a task as completed
- Delete a task
- Exit the app
